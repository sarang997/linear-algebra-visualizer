What are vectors?

Vectors are mathematical objects that have both magnitude (length) and direction. They are fundamental to linear algebra and machine learning.

Key Vector Operations:
  • Addition: Combining two vectors (tip-to-tail method)
  • Subtraction: The difference between two vectors
  • Dot Product: A scalar value representing how parallel two vectors are
  • Cross Product: A vector perpendicular to both input vectors

In machine learning, vectors represent features, weights, and gradients. Understanding vector operations helps in grasping concepts like gradient descent.

Loss Function

Mean Squared Error (MSE)

The loss function measures how well our model is performing. For regression problems, we typically use Mean Squared Error (MSE), which calculates the average squared difference between predicted and actual values.
